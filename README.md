# nodejs-starter

## Setup

1. Install dependencies

```bash
npm install
```

2. Create a `.env` file and fill the values

```bash
cp .env.template .env
```

3. Start the server

```bash
npm run start:dev
```

## Build

```bash
npm run build
```

## Start the built server

```bash
npm run pm2:start
```

## Stop the built server

```bash
npm run pm2:stop
```
